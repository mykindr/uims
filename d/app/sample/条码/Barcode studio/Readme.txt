1. 
What is Barcode studio ?

Barcode studio is set of components for easy create, design, print and export barcode from Borland/Embarcadero Delphi and C++ Builder environments.

2.
Main features :
-------------------------------------------------------------------------------------
- Support for Delphi 7-Delphi XE, C++Builder.

- most used linear symbologies 
...... EAN 8/13, UPC-A, UPC-E, ISBN, JAN (all with 2 or 5 aditional digits), Code39, Code93, Code128, Code 2 of 5 (Industrial, Interleaved, Datamatrix, IATA, Invert), Codabar, Code11, OPC, 
...... Postal symbologies : PostNet, RoyalMail, Dutch4StatePostal, 4 State, Singapore4StatePostalCode, Planet, SwissPostal , Postbar, Telepen, FIM
.....  Stacked symbologies PDF417, PDF417 Truncated, PDF417 Macro, PDF417 Micro
.....  2D symbologies DataMatrix and QR Code



3. Supported report engines
----------------------------------------------------------------------------
........ FastReport  ( http://www.fast-report.com/ )
........ Quick Report (http://www.qusoft.com/ )
........ RAVE report  ( included in latest Delphi/C++Builder versions)
........ Report Builder ( http://www.digital-metaphors.com/ )
........ ACE reporter ( http://www.sct-associates.com/ ) 
........ direct print to any device using TCanvas , without any report engine, direct Barcode studio methods call
